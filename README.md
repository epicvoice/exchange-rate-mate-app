# Exchange Rate Mste App
Exchange Rate Mate is a modern and simple web app to check the latest currency exchange rates. Convert between PLN, USD, EUR, CHF, and GBP quickly and easily.


> ⚠️ **Note:** The JavaScript functionality for swapping currencies and fetching live rates is planned and will be added soon.

## Features

- Responsive, modern design
- Easy-to-use currency converter interface
- Swap currencies (coming soon)
- Live currency rates (coming soon)
- Works on desktop and mobile devices
